# 🎬 Netflix Content Analysis using Python

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the Netflix Titles dataset using Python. The objective is to uncover insights about Netflix's content library, including content types, genres, release trends, country-wise distribution, ratings, and durations through data cleaning, visualization, and statistical analysis.

---

## 📊 Dataset

* **Dataset:** Netflix Titles Dataset
* **Source:** Kaggle
* **Format:** CSV

---

## 🎯 Project Objectives

* Clean and preprocess the dataset
* Analyze Movies vs TV Shows
* Explore content distribution across countries
* Identify the most common genres
* Analyze yearly content additions and release trends
* Study content ratings and durations
* Generate meaningful business insights using visualizations

## 💡 Key Insights

* Movies make up a larger portion of Netflix's catalog than TV Shows.
* The United States contributes the highest amount of content.
* Content additions increased significantly after 2015.
* Drama, International Movies, and Comedies are among the most common genres.
* TV-MA is one of the most frequently assigned content ratings.
